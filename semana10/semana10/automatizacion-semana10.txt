Proceso a automatizar: Envío de recordatorios de tareas vencidas

Herramienta: GitHub Actions + Python

Pasos:
1. Script en Python revisa fechas de tareas cada día a las 8am
2. Si hay tareas vencidas, envía correo al responsable
3. Se ejecuta automático con GitHub Actions

Beneficio: Evita que se olviden tareas y no se necesita hacerlo manual.
